In this asset, you will find 4 scripts:
CarTrafficLightGreen
CarTrafficLightRed
PeopleTrafficLightGreen
PeopleTrafficLightRed
that are attached to traffic lights.

These scripts change materials at traffic lights to create a real switching effect.
To change the emission time, change the values in the inspector using the variables "Red Time" and "Green Time"
